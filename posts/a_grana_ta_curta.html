A grana tá curta? A gente te ajuda a fazer o controle

para sobrar dinheiro para breja.

A fase de iniciar uma faculdade em uma outra cidade é uma barra para qualquer um, morar sozinho e ter responsabilidades de pagar as contas e se virar com a grana que tem ou ganha ... é um o momento difícil. Mas segue algumas dicas importantes para controlar os gastos e sobrar uma graninha.

Segue um roteiro prático que vai te ajudar a organizar as contas:

 Defina todas as suas despesas;
 Estabeleça prioridades;
 Faça a divisão 70 / 20 / 10, sendo 70% para contas fixas, 20% para lazer e 10% para reserva;
 Valide todos os meses se a divisão acima está sendo respeitada;
 Revise o plano se necessário. Primeiro defina os valores gastos para cada finalidade

Segue um exemplo para você entender a divisão: Se você recebe
R$ 2.200,00 a divisão ficará a seguinte:
 R$ 1.540,00 para contas fixas;
 R$ 440,00 para lazer;
 R$ 220,00 para reserva. Com essa divisão você já tem o dinheiro reservado para cada gasto, e conforme os imprevistos acontecem você pode usar a reserva quando necessário.
